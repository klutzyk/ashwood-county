# Ashwood County

Ashwood County is the working title of a top-down 2D zombie-survival RPG focused on exploration, scarce resources, difficult decisions, and believable survivor relationships.

The zombies create pressure.

The survivors create stories.

## Status

Early pre-production and prototyping.

The project is not currently playable.

## Core Direction

The intended game combines:

- Top-down survival and exploration
- Handcrafted locations
- Looting and resource management
- Zombie encounters
- Named survivors with goals and relationships
- Structured memories of important player choices
- Character-driven situations
- A changing community
- Environmental storytelling
- A restrained central mystery or narrative thread

All features remain subject to prototyping and scope review.

## Technology

- Godot 4 .NET
- C#
- Compatibility renderer
- Windows desktop target
- Cursor and Codex
- Git and GitHub

## Current Milestone

The first vertical slice will allow the player to:

1. Move through a small environment
2. Search a container
3. Collect medicine
4. Return to one survivor
5. Decide whether to give them the medicine
6. Change that survivor's trust
7. Record the decision as structured memory
8. Save and load the result

## Development Principles

- Build small complete slices
- Prioritise gameplay over feature count
- Prefer connected systems over shallow content
- Keep authoritative game state deterministic
- Target modest hardware
- Avoid premature procedural generation
- Keep generated dialogue separate from game-state authority
